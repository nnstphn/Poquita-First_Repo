My Projects

Project Name Description Status Date Completion
STAMP: Scanning Tactile and Audio Multi-Mode Platform a portable, offline device designed to help blind users read printed text in real time. STAMP provides three output modes— Braille-only, Audio-only, or a Combined Braille and Audio option On-going 2026
Hand Gesture Vehicle using Arduino is an Arduino-based robotic car that moves according to the user’s hand movements. Using a motion sensor like an MPU6050, the system detects hand tilts and sends commands wirelessly to control the vehicle’s direction. This allows smooth and intuitive control without using buttons or joysticks. Completed 2025
Temperature Sensor using ESP32 with SMS is a system that monitors temperature in real time and sends SMS alerts through an internet-based SMS service (API) instead of a GSM module. The ESP32 reads data from the temperature sensor and, when a set limit is reached, it connects to Wi-Fi and sends a message to the user, allowing remote monitoring and quick response to temperature changes. Completed 2024
